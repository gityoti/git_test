# markdown 语法学习

|描述|效果|语法|
|:--|:--:|--:|
|单行代码|这是一段文字`ABC`描述|\`突出的文字\`|、
|斜体|*斜体*|`*斜体*`|
|斜体|_斜体_|`_斜体_`|
|加粗|**加粗**|`**加粗**`|
|加粗+斜体|***加粗+斜体***|`***加粗+斜体***`|
|加粗+斜体|**_加粗+斜体_**|`**_加粗+斜体_**`|
|删除线|~~删除线~~|`~~删除线~~`|
